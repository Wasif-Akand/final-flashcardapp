Getting Started
1.Create a Subject → add a Topic inside it
2.Select the topic from the sidebar — you'll see the Lessons screen
3.+ New Top-Level Lesson → pick Flashcards, MCQ, or SAQ, then add cards from the Data tab
4.Click a lesson to start reviewing — MCQ lessons are tap-to-answer, SAQs show the answer and mark, all tracked by SRS
5.Use the ← Back button to return to lesson list anytime
Import format: Flashcards → Question -!- Answer  ·  MCQ → Question -!- Option 1 -!- Option 2 -!- Option 3 -!- Option 4 -!- Correct Answer  ·  SAQ → Question -!- Mark
